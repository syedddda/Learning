Flex Direction

Flex directions of row-reverse and column-reverse will actually reverse the items only visually and the position of them in the DOM tree is never changed. They will also appear in the DOM order for any screen reading and keyboard tabbed navigation

flex-direction: row

flex-direction: row-reverse

Row reverse actually reverses the items but also reverses the start and end. Below you can see that the order of 1,2,3 is reversed but they don't start from the left. Instead, they start from the right. This is the same for column reverse too!

Flex Order

Flex order is used to position the flex children inside a flex parent. Atleast two flex children should have an order specified for the order to take effect. When the order is specified for two children, those children that were not given an order will be displayed first in the semantic order and then those with the order specified will be displayed in the specified order(Lowest to highest)

If the same order number is given for two children, then the one that is semantically appearing first in the Html will be shown first
